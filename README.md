### Drum Kit Web Application

---

### Overview
Drum Kit is an interactive web application that allows users to play different drum sounds using keyboard keys or mouse clicks.

This project demonstrates fundamental concepts of:
- HTML structure
- CSS styling
- JavaScript event handling
- DOM manipulation
- Audio integration in web applications

---

### Features
- Play drum sounds using keyboard keys
- Click on drum buttons to produce sounds
- Visual button animation on interaction
- Responsive and interactive UI

---

### Tech Stack
- HTML5
- CSS3
- JavaScript

### How It Works
- Each drum button is mapped to a specific keyboard key.
- When a key is pressed or a button is clicked:
  - JavaScript detects the event.
  - The corresponding sound file from the `sounds/` folder is played.
  - A visual animation is triggered using CSS.

### How to Run

1. Clone the repository:

git clone https://github.com/Shrestha771/Drum-Kit.git  
cd drum-kit  

2. Open the project:

- Navigate to `Drum Kit Starting Files`
- Open `index.html` in your browser

No installation required.
